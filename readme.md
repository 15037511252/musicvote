# 点歌台助力V1.0

## 1.添加本地歌曲

### 1.1 在根目录把歌单创建文件music.txt

要求music.txt格式如下:

```txt
红色高跟鞋-蔡健雅 
刚好遇见你-李玉刚 
暧昧-薛之谦 
男孩-梁博
```

注意：每一行中歌名和歌手之间用横杠‘-’隔开，如果歌名或者歌手名字出现横杠需要去掉，否则录入不完整。

### 1.2在地址栏访问http://localhost/music/add

录入歌曲的功能只能在本地是用，当录入成功会返回` 成功录入31首歌曲 `



## 2.访问歌单

访问地址：http://域名/
